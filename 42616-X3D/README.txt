Para ser possível ver as texturas, é necessário ter o python3 atualizado, e abrir o terminal na pasta do trabalho, para poder correr a linha de comando "python -m SimpleHTTPServer 42069" (este valor inteiro não precisa necessáriamente de ser este, este era o valor que o aluno usava para testar, mas precisa ser o mesmo que no link a sequir) e deixar aberto, logo de seguida é necessário abrir um link do server criado "http://localhost:42069". Caso isto não seja feito, não vai ser possível observar nada, já que sem texturas todas as figuras vão ficar com a cor preta.
